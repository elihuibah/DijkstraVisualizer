## Descripción
Aplicación GUI de Java que permite crear nodos (o vértices) y conectarlos con aristas para visualizar las rutas más cortas entre dos nodos seleccionados utilizando el algoritmo de Dijkstra.

## Funcionalidades
Importa el proyecto de Java a tu IDE de preferencia. Al ejecutarse, se despliega una interfaz gráfica en la que el usuario crea grafos con los nodos y vértices que agregue.

**MODOS DE INTERACCIÓN:**

  **- Añadir Nodo**: Haz clic en el panel central para crear nuevos nodos (círculos).
  
  **- Añadir Aristas**: Haz clic en un nodo y luego en otro. Se te pedirá ingresar el **peso** (ponderación) de la arista.
  
  **- Mover Nodo**: Haz clic y arrastra un nodo para reubicarlo en el panel.

**ALGORITMOS Y HERRAMIENTAS:**

  **- Caminos Mínimos**: Calcula el camino más corto entre dos nodos que tú elijas con el algoritmo de Dijkstra. Muestra la ruta y el costo total.
  
  **- Rutas Alternativas**: Calcula y muestra hasta 5 rutas diferentes entre dos nodos, listándolas en el panel inferior.
  
  **- Añadir Todas las Aristas**: Conecta cada nodo con todos los demás, asignando pesos aleatorios.
  
  **- Peso Aleatorio**: Asigna un nuevo peso aleatorio a todas las aristas existentes.
  
  **- Eliminar Nodo**: Abre un diálogo para seleccionar un nodo por su ID y eliminarlo (y sus aristas conectadas).
  
  **- Eliminar Arista**: Abre un diálogo para seleccionar una arista por sus nodos de conexión y eliminarla.
  
  **- Limpiar Grafo**: Elimina todos los nodos y aristas del panel.
  
  **- Modo Claro / Oscuro**: Cambia el esquema de colores de la aplicación.
